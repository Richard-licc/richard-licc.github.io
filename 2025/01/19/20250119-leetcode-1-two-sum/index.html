Leetcode 解題 - 1. Two Sum

發表於2025-01-19|更新於2025-01-20|理查刷 Leetcode
|瀏覽量:

題目

給定一個 array 以及一個 target number，查找 array 中，哪兩個 number 相加可以等於 target number

題目保證一定且只有一組解存在

解法

Two Sum 可以說是最經典的 leetcode 題了

法一：暴力解

最直覺的方式就是兩層迴圈去遍歷 array 中所有 number，看哪組配對可以相加後等於 target number

Time Complexity: O(N²) → 兩層迴圈的時間複雜度Space

Complexity: O(1) → 不考慮一開始給定的 array 空間的話

法二：Hash Map

另一種方式則是用 hash map

用一層迴圈遍歷 array 中的數值，針對每個數值來記錄資料到 hash map 中，key 為 “target number — current value”，value 為 “current index”

當後續的數值為 hash map 的 key 時，就可以跟 array index 為 hash map value 的 element 配對組成答案。

而後每次遍歷到新的 element 時，先查找 map 中 key 為 current value 的欄位有沒有值，有的話代表可以跟先前的元素配對作為答案，皆大歡喜世界和平，沒有的話就紀錄當前的資料，再繼續往下遍歷

Time Complexity: O(N) → array 跑一遍就可以找到答案了

Space Complexity: O(N) → 最壞的情況需要存下 N-1 個紀錄到 map 中

Code (Hash Map)
